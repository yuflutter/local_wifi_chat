# Local WiFi Chat

Текстовый и голосовой чат, работающий в локальной сети без доступа к интернету. Бекенд написан на golang, компилируется в один бинарник (для каждой целевой платформы), который можно запустить на WiFi-роутере или на смартфоне (через Termux). При запуске поднимается http-сервер, по которому бекенд отдаёт файлы фронтенда, а также предоставляет эндпойнты для API. Фронтенд имеется в двух вариантах - на Flutter и Compose Multiplatform. Чат хранится в памяти бекенда (информация удаляется при остановке процесса), на клиентских браузерах хранится лишь информация о текущем пользователе. Проект сделан с целью сравнить Flutter vs Compose MP.
